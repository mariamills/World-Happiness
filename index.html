World Happiness Report 2023

The World Happiness Map is an interactive data visualization that maps out the happiness scores, GDP per capita, and social support for various countries around the world.

The chart uses emoji symbols to represent the happiness score for each country, with a smiley face for scores above 5 and a sad face for scores below 5. Users can filter the data by country and explore the relationships between GDP per capita, social support, and happiness score.

The chart is built using D3.js and is based on the World Happiness Report dataset 2023.
